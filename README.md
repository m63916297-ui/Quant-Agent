# Quant AI - Multi-Agent Trading System

Sistema multiagente cuantitativo para trading y gestión de portafolios.

## Agentes

- **Portfolio Agent**: Gestión y optimización de portafolios
- **Trading Agent**: Señales de trading y backtesting
- **Market Agent**: Análisis de mercado y detección de regímenes
- **Risk Agent**: Gestión de riesgos y VaR

## Instalación

```bash
pip install -r requirements.txt
```

## Uso

```bash
python main.py
```

## Frontend Streamlit

```bash
streamlit run frontend/app.py
```
